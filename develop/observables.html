Observables¶

class qucumber.observables.Observable[source]¶

Base class for observables.

apply(samples, rbm)[source]¶

Computes the value of the observable, row-wise, on a batch of samples. Must be implemented by any subclasses.

Parameters:
  • samples (torch.Tensor) – A batch of samples to calculate the observable on.
  • rbm (qucumber.rbm.BinomialRBM) – The RBM that drew the samples.
sample(rbm, num_samples, **kwargs)[source]¶

Draws samples of the observable using the given RBM.

Parameters:
  • rbm (qucumber.rbm.BinomialRBM) – The RBM to draw samples from.
  • num_samples (int) – The number of samples to draw.
  • **kwargs – Keyword arguments to pass to the RBM’s sample function.
expected_value(rbm, num_samples, batch_size=0, **kwargs)[source]¶

Estimates the expected value of the observable over the distribution defined by the RBM.

In order to avoid running out of memory, the expected value computation can be performed in batches.

Parameters:
  • rbm (qucumber.rbm.BinomialRBM) – The RBM to draw samples from.
  • num_samples (int) – The number of samples to draw.
  • batch_size – The size of the batches; if 0, will use one batch containing all drawn samples.
  • **kwargs – Keyword arguments to pass to the RBM’s sample function.
Returns:

The estimated expected value of the observable.
Return type:

float
variance(rbm, num_samples, batch_size=0, **kwargs)[source]¶

Estimates the variance (using the sample variance) of the observable over the distribution defined by the RBM.

In order to avoid running out of memory, the variance computation can be performed in batches.

Parameters:
  • rbm (qucumber.rbm.BinomialRBM) – The RBM to draw samples from.
  • num_samples (int) – The number of samples to draw.
  • batch_size – The size of the batches; if 0, will use one batch containing all drawn samples.
  • **kwargs – Keyword arguments to pass to the RBM’s sample function.
Returns:

The estimated variance of the observable.
Return type:

float
statistics(rbm, num_samples, num_chains=0, burn_in=1000, steps=1)[source]¶

Estimates both the expected value and variance of the observable over the distribution defined by the RBM.

Parameters:
  • rbm (qucumber.rbm.BinomialRBM) – The RBM to draw samples from.
  • num_samples (int) – The number of samples to draw. The actual number of samples drawn may be slightly higher if num_samples % num_chains != 0.
  • num_chains (int) – The number of Markov chains to run in parallel; if 0, will use a number of chains equal to num_samples.
  • burn_in (int) – The number of Gibbs Steps to perform before recording any samples.
  • steps (int) – The number of Gibbs Steps to take between each sample.
Returns:

A dictionary containing both the (estimated) expected value (key: “mean”) and variance (key: “variance”) of the observable.
Return type:

dict(str, float)
