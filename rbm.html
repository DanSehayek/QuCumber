Real Positive Wavefunction RBM¶

class qucumber.rbm.rbm_vanilla.RBM(num_visible, num_hidden, gpu=True, seed=1234)[source]¶

Class to build the Restricted Boltzmann Machine.

Variables:
  • num_visible (int) – Number of visible units (determined from the input training data).
  • num_hidden (int) – Number of hidden units to learn the amplitude (default = num_visible).
  • gpu (bool) – Should the GPU be used for the training (default = True).
  • seed (int) – Fix the random number seed to make results reproducable (default = 1234).
Parameters:
  • weights (torch.doubleTensor) – The weight matrix for the amplitude RBM (dims = num_hidden x num_visible). Initialized to random numbers sampled from a normal distribution.
  • visible_bias (torch.doubleTensor) – The visible bias vector for the amplitude RBM (size = num_visible). Initialized to zeros.
  • hidden_bias – The hidden bias vector for the amplitude RBM (size = num_hidden). Initialized to zeros.
Raises:

ResourceWarning – If a GPU could not be found; continue running the program on the CPU.

effective_energy(v)[source]¶

The effective energies of the given visible states.

\mathcal{E}(\bm{v}) &= \sum_{j}b_j v_j + \sum_{i}\log \left\lbrack 1 + \exp\left(c_{i} + \sum_{j} W_{ij} v_j\right) \right\rbrack

train(data, epochs, batch_size, k=10, persistent=False, lr=0.001, momentum=0.0, l1_reg=0.0, l2_reg=0.0, initial_gaussian_noise=0.0, gamma=0.55, callbacks=[], progbar=False, starting_epoch=0)[source]¶

Execute the training of the RBM.

Parameters:
  • data (array_like of doubles) – The actual training data
  • epochs (int) – The number of parameter (i.e. weights and biases) updates (default = 100).
  • batch_size (int) – The size of batches taken from the data (default = 100).
  • k (int) – The number of contrastive divergence steps (default = 10).
  • persistent (bool) – Whether to use persistent contrastive divergence (aka. Stochastic Maximum Likelihood) (default = False).
  • lr (double) – Learning rate (default = 1.0e-3).
  • momentum (double) – Momentum hyperparameter (default = 0.0).
  • l1_reg (double) – L1 regularization hyperparameter (default = 0.0).
  • l2_reg (double) – L2 regularization hyperparameter (default = 0.0).
  • initial_gaussian_noise (double) – Initial gaussian noise used to calculate stddev of random noise added to weight gradients (default = 0.01).
  • gamma (double) – Parameter used to calculate stddev (default = 0.55).
  • callbacks ([Callback]) – A list of Callback functions to call at the beginning of each epoch
  • progbar (bool or "notebook") – Whether or not to display a progress bar. If “notebook” is passed, will use a Jupyter notebook compatible progress bar.
  • starting_epoch – Which epoch to start from. Doesn’t actually affect the model, exists simply for book-keeping when restarting training.
Returns:

None
